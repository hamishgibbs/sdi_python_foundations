Resources

Author

Hamish Gibbs

Python

  • The Python Tutorial: the official python tutorial. Can be quite detail-heavy, but it is also a great reference.
  • Python For Everyone: the most popular course for learning Python. This course has lots of great resources and exercises.
  • Real Python: lots of in-depth articles and interesting Python projects. They also offer paid content. I always recommend sticking to the free stuff!

Git

  • Git Guides (GitHub): a useful resource for basic git commands and installation instructions.
  • Cheatsheet: Resolving Conflicts in a Git Repository: my own cheatsheet on how to fix a common issue for people learning git and working in shared repositories.
  • Visualizing Git: a fun tool to visualize how git works. Playing around with this can help you “get” how different commands interact with each other.

Personal projects

I recommend finding personally compelling reasons to learn Python, beyond what is available in online tutorials.

Here are a few of the small projects I have made to learn about different aspects of Python programming:

  • https://github.com/hamishgibbs/lit_review_graph
    • An app to visualize the citation network connecting academic publications.
  • https://github.com/hamishgibbs/meal_prep_ai
    • The backend to an AI chatbot that can make my weekly grocery shopping list.
  • https://github.com/hamishgibbs/yui
    • A very simple command line todo list.
  • https://github.com/hamishgibbs/scrape_transfermarkt
    • A web scraper to download football match attendance counts.
